### 指派：所有用户都可指派

- `getAssignableById`：只查 user 存在，不检查 is_active / 邀请门

之前 `#` 列是「当前页 + index」按计算式生成的临时序号，翻页/筛选后会变，沟通时引用「第 3 条」毫无意义。

改成显示 report.id 的前 8 位 hex（如 `628c9c21`）：
- **稳定**：跟 issue 一一对应，沟通时可以直接引用 ID
- **点击复制**：单击 cell 复制完整 `rpt_xxx` 到剪贴板，toast 提示
- **hover 显示完整 ID**（HTML title 属性）
- **等宽字体**：font-mono + tabular-nums
