# Changelog 1.0.23

去掉「指派给我」特殊项，自己也可以直接从用户列表选（lula 2026-06-01）。

## 🔄 Changes

### 指派列简化

v1.0.21 引入的「指派给我」`SelectItem` + `ASSIGN_TO_ME` sentinel 改造为：

- 直接列出所有 `assignableUsers`（含当前登录用户）
- 自己的名字后加「（我）」/「(you)」标注，便于识别
- 删除 `ASSIGN_TO_ME` sentinel 和相关 fallback 逻辑
- `handleAssigneeChange` 简化为：`value === UNASSIGNED ? null : value`

**为什么**：v1.0.21/22 上「指派给我」反复 400，根因是 sentinel 转 user.id 在某些 race 下失败。后端 `findAssignable` 本来就不排除当前用户，没必要再造一个特殊项。直接列表选最稳。
